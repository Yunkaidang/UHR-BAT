# UHR-BAT LongVA Branch: with_k-means

This branch contains the LongVA-based implementation used for K-Means region grouping and multiscale attention selection in UHR-BAT.

Install from this directory when you want to run the K-Means scripts:

```bash
pip install -e . --no-deps
```

See the repository root `README.md` for full environment, data, training, and evaluation instructions.
