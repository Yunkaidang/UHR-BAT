## 🔍 Introduction

Ultra-high-resolution remote sensing images contain rich visual details but also introduce significant computational challenges for vision-language models.

In this work, we propose **UHR-BAT**, a **budget-aware token compression vision-language model** for efficient and effective understanding of ultra-high-resolution remote sensing imagery.

UHR-BAT reduces redundant visual tokens while preserving task-relevant information, enabling vision-language models to process large-scale remote sensing images under limited computational budgets.

## ✨ Highlights

- 🛰️ Designed for **ultra-high-resolution remote sensing** image understanding.
- 🧩 Introduces **budget-aware token compression** for vision-language models.
- ⚡ Reduces visual token redundancy and computational cost.
- 🎯 Preserves task-relevant and semantically important visual information.
- 📈 Achieves strong performance under constrained token budgets.
