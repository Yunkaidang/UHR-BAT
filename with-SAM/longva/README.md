# UHR-BAT LongVA Branch: with-SAM

This branch contains the LongVA-based implementation used for SAM-guided multiscale token-mask training and evaluation in UHR-BAT.

Install from this directory when you want to run the SAM/mask-based scripts:

```bash
pip install -e . --no-deps
```

See the repository root `README.md` for full environment, data, training, and evaluation instructions.
